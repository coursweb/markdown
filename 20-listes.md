Une liste à puces:

```
- First item
- Second item
- Third item
- Fourth item
```

Résultat:

- First item
- Second item
- Third item
- Fourth item

Avec indentation:

```
- First item
- Second item
- Third item
  - Indented item
  - Indented item
- Fourth item
```

Résultat:

- First item
- Second item
- Third item
  - Indented item
  - Indented item
- Fourth item
